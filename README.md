Dependecies install
We use poetry as dependecies manager. First install poetry using
1. `pip install poetry`
Than acitvate venv:
2. `poetry shell`
Than install all dependecies:
3. `poetry instlal`
To add new dependecies use:
`poetry add <dep_name>`
