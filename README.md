### Dependecies install

We use poetry as dependecies manager.

First install poetry using

1. `pip install poetry`

Then acitvate venv:

2. `poetry shell`

Then install all dependecies:

3. `poetry install`

To add new dependecies use:

`poetry add <dep_name>`

### Run the app

To run the app execute this command in `src/` folder:

`fastapi dev main.py`
